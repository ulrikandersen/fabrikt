# Security Policy

## Scope

Fabrikt is a build-time code generation tool — it is not deployed as a runtime service and does not handle sensitive data in production environments. Security vulnerabilities are therefore low severity by default.

## Supported Versions

The latest release on [Maven Central](https://central.sonatype.com/artifact/com.cjbooms/fabrikt) is the only supported version.

## Reporting a Vulnerability

If you discover a security issue, please report it privately via [GitHub Security Advisories](https://github.com/fabrikt-io/fabrikt/security/advisories/new) rather than opening a public issue.

We'll respond within a reasonable timeframe given this is a community-maintained project.
